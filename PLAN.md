# MBTI Dating Bot — План разработки

## Обзор
Telegram-бот для знакомств в тематике MBTI.
Аудитория: существующее VK-сообщество (472 человека).
Цель: MVP → рост → монетизация.

---

## Технический стек

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.11+ |
| Фреймворк бота | aiogram 3.x |
| База данных | PostgreSQL (Neon free tier) |
| Кеш/сессии | Redis (Upstash free tier) |
| Хранение фото | Telegram File API (file_id в БД) |
| Хостинг | Render (free tier) |
| Платежи | Telegram Stars |

**Итого стоимость на старте: 0₽/мес**

---

## Фаза 1: MVP (1-2 недели)

| День | Задача |
|------|--------|
| 1-2 | Структура проекта, БД (PostgreSQL), подключение aiogram |
| 3-4 | Создание анкеты: MBTI, возраст, город, описание, 1-3 фото |
| 5-6 | Просмотр анкет: листание, базовые фильтры (MBTI, город) |
| 7 | Лайк/дизлайк + мэтч при взаимном лайке |
| 8 | Управление своей анкетой (редактирование, скрыть/показать) |
| 9 | Админ-панель: уведомления о новых анкетах, approve/reject |
| 10 | Деплой на Render, тесты |

---

## Фаза 2: Социальные фичи (1 неделя)

| День | Задача |
|------|--------|
| 1-2 | Отзывы к анкетам (текст + оценка 1-5) |
| 3 | Рейтинг анкет (средняя оценка + количество лайков) |
| 4 | Жалобы на анкеты → уведомление админу |
| 5 | Статистика бота (сколько анкет, мэтчей, активных) |
| 6-7 | Полировка, багфиксы |

---

## Фаза 3: Премиум + рефералка (1 неделя)

| День | Задача |
|------|--------|
| 1-2 | Премиум-подписка (Telegram Stars) |
| 3 | Премиум-фичи: безлимит просмотров, "кто лайкнул меня", boosted видимость, расширенные фильтры |
| 4-5 | Реферальная система: инвайт-линк, отслеживание, начисление премиума |
| 6 | Реклама в боте (разрешённые посты, раз в N дней) |
| 7 | Тесты, полировка |

## Деплой на Render

1. Создать Web Service на Render
2. Подключить репозиторий GitHub
3. Добавить переменные окружения
4. Подключить PostgreSQL (free tier)
5. Настроить health check

### Переменные окружения

```
BOT_TOKEN=your_telegram_bot_token
DATABASE_URL=postgresql://...
REDIS_URL=redis://...
ADMIN_IDS=123456789,987654321
PREMIUM_PRICE_STARS=299
REFERRAL_BONUS_DAYS=3
```
