# MBTI Dating Bot — План разработки

## Обзор
Telegram-бот для знакомств в тематике MBTI.
Аудитория: существующее VK-сообщество (472 человека) + Telegram-чат.
Цель: MVP → рост → монетизация.

---

## Технический стек

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.11+ |
| Фреймворк бота | aiogram 3.x |
| Мини-приложение | Telegram Web App (HTML/JS) |
| Backend | FastAPI + Uvicorn |
| База данных | PostgreSQL (Neon free tier) |
| Кеш/сессии | Redis (Upstash free tier) |
| Хранение фото | Telegram File API (file_id в БД) |
| Хостинг | Render (free tier) |
| Платежи | Telegram Stars |

**Итого стоимость на старте: 0₽/мес**

---

## Текущее состояние

- Бот работает на Render (webhook)
- Mini App открывается, но UI минимальный
- БД: SQLite (нужно мигрировать на PostgreSQL)
- API-роуты подключены, auth через `X-Telegram-Init-Data` работает
- Нет: фото, мэтчей, тегов, inline-режима, премиума, рефералки

## Фаза 0: Фундамент UI (1-2 недели)

### 0.1 Стартовый экран — «Мой профиль»
**Приоритет: 🔴 Критично**

Mini App открывается с экрана профиля пользователя:
- Фото профиля (или заглушка если нет)
- Имя, возраст, MBTI
- Статус анкеты: «Создана» / «На модерации» / «Опубликована» / «Скрыта»
- Блок статистики: лайки, мэтчи, рейтинг, отзывы
- Кнопки навигации: «Листать анкеты», «Редактировать», «Мэтчи»
- Нижний блок: «Премиум», «Рефералы», «Настройки»

Для нового пользователя (нет анкеты):
- Приветствие + кнопка «Создать анкету»
- Краткое описание что это за бот

### 0.2 Пошаговое создание анкеты (Onboarding)
**Приоритет: 🔴 Критично**

Разбить на шаги с прогресс-баром:
1. **Фото** — загрузка 1-5 фото из галереи (через Telegram WebApp `chooseMedia` или `showPopup`)
2. **Основное** — имя, пол, возраст
3. **MBTI** — сетка 16 типов с кратким описанием каждого
4. **О себе** — форма общения, описание, кого ищет
5. **Теги интересов** — выбор до 7 из предустановленного списка
6. **Музыка** — любимые исполнители/группы (текстовое поле)
7. **Правила** — модальное окно с правилами + чекбокс «Согласен»

Правила при публикации:
- Реальные фото, без 18+ контента
- Не спамить, не оскорблять
- Люди не психологи — не заполняйте людьми свою пустоту
- Не ебьте людям мозги
- За нарушения — бан без предупреждения

### 0.3 Улучшенная карточка браузинга
**Приоритет: 🔴 Критично**

Карточный интерфейс (Tinder-like):
- Фото на весь экран карточки (свайп влево/вправо или кнопки)
- Имя, возраст поверх фото
- MBTI тип + процент совместимости (из матрицы `utils/mbti.py`)
- Теги интересов (иконки/бейджи)
- Музыкальные предпочтения
- Описание (раскрывается по тапу)
- Кнопки: 💬 Комментарий, ❤️ Лайк, ✖️ Пропустить
- Счётчик оставшихся просмотров (если не премиум)
- Фильтры: MBTI, возраст, пол, теги

### 0.4 Экран мэтчей
**Приоритет: 🔴 Критично**

## Фаза 1: Социальные фичи (1 неделя)

### 1.1 Комментарии к анкетам
**Приоритет: 🟡 Важно**

- Бесплатно: комментировать анкеты (лимит 5/день)
- Комментарий виден владельцу анкеты
- Уведомление ботом: «Кто-то прокомментировал твою анкету»
- Просмотр комментариев в разделе «Активность»

### 1.2 Отзывы и рейтинги
**Приоритет: 🟡 Важно**

- После мэтча — предложение оставить отзыв (1-5 звёзд + текст)
- Отзыв отображается в профиле пользователя
- Средняя оценка в статистике
- Нельзя изменить отзыв (только дополнить)

### 1.3 Жалобы
**Приоритет: 🟡 Важно**

- Кнопка «Пожаловаться» на профиле
- Причины: спам, оскорбления, фейк, 18+, другое
- Жалоба → уведомление админу
- При 3+ жалобах — автоматическая скрытие анкеты до модерации

### 1.4 Inline-режим для Telegram-чата
**Приоритет: 🟡 Важно**

- `@mbtidatebot username` — показывает карточку профиля пользователя прямо в чате
- Кнопка «Открыть Mini App» под карточкой
- Если у пользователя нет анкеты — «У пользователя ещё нет анкеты»
- Работает в любом чате где есть бот
- Виральный рост: люди из чата видят бота и заходят

### 1.5 «Вопрос дня»
**Приоритет: 🟢 Nice to have**

- Бот присылает вопрос раз в день (через cron/notification)
- Примеры: «Какой MBTI тип тебя бесит?», «Красный флаг в отношениях?»
- Ответ отображается в профиле пользователя
- Ice-breaker для первого сообщения после мэтча

### 2.1 Премиум-подписка
**Приоритет: 🟡 Важно**

Оплата через Telegram Stars (299 Stars/мес).

**Бесплатно:**
- Создание и редактирование анкеты

- Базовые фильтры (MBTI, пол, возраст)

**Премиум:**
- 👻 **Режим невидимки** — просматривать анкеты не появляясь у других
- 👀 **Кто лайкнул меня** — полный список (не только мэтчи)
- 🎯 **Приоритет в ленте** — профиль показывается чаще (x2-3)
- 📊 **Расширенная статистика** — кто заходил на профиль, графики активности
- 💬 **Безлимитные комментарии** — без дневного лимита
- 🔍 **Расширенные фильтры** — по тегам, музыке, городу
- ⭐ **VIP-бейдж** — выделение профиля в ленте
- 👥 **Групповые анкеты** — создание анкет «ищу компашку»
- 📢 **Анкеты каналов** — реклама своего TGK
- 🎉 **Ивенты** — создание событий для групп

### 2.2 UI Премиума
**Приоритет: 🟡 Важно**

- Экран «Премиум» с описанием фич
- Кнопка «Купить за 299 Stars» (через `requestInvoice`)
- Статус подписки + дата окончания
- Визуальное отличие премиум-профилей (бейдж, рамка)

### 2.3 Реферальная система
**Приоритет: 🟡 Важно**

- Реферальная ссылка: `https://t.me/mbtidatebot?start=12345`

- Бонус: 3 дня премиума за каждого друга с созданной анкетой
- Уведомление: «Твой друг создал анкету! +3 дня премиума»

### 2.4 Рекламные кампании
**Приоритет: 🟢 Nice to have**

- Админ создаёт рекламный пост в админке

## Фаза 3.5: Группы и каналы (1 неделя)

### 3.5.1 Типы анкет
**Приоритет: 🟡 Важно**

Три типа анкет с разной логикой:

| Тип | Доступ | Описание |
|-----|--------|----------|
| **Личная** | Бесплатно | Дейтинг, знакомства 1-на-1 |
| **Групповая** | Премиум | «Ищу компашку», «Набираю группу по интересам» |
| **Канал** | Премиум | «У меня есть TGK, приходите» — мини-реклама |

### 3.5.2 Групповые анкеты
**Приоритет: 🟡 Важно**

Структура групповой анкеты:
- 📛 Название группы (например «Настолки по субботам»)
- 👤 Организатор — привязка к личному профилю (ENFJ, 25)
- 🏷️ Теги — интересы группы (Настолки, Оффлайн, Москва)
- 📝 Описание — «Ищем 4-6 человек для регулярных игр...»
- 👥 Сколько ищут — целевой размер группы
- 👥 Уже в группе — текущее количество участников
- 📅 Формат — Оффлайн / Онлайн / Гибрид
- 📍 Город (для оффлайн)

Логика:
- Лайк на групповую анкету = «Хочу присоединиться»
- Организатор получает уведомление: «Кто-то хочет в вашу группу»
- Организатор принимает/отклоняет заявку
- Принятые участники видят друг друга → могут мэтчиться между собой
- Лимит: 1 активная групповая анкета на пользователя

### 3.5.3 Анкеты каналов (TGK)
**Приоритет: 🟡 Важно**

Структура анкеты канала:
- 📛 Название канала
- 👤 Владелец — привязка к личному профилю
- 🏷️ Тематика — теги (Стоицизм, Программирование, MBTI)
- 📝 Описание — «Показываю свою деятельность X, сидят люди по интересам Y»
- 🔗 Ссылка — `https://t.me/channel_name`
- 👥 Подписчиков из сообщества — сколько участников бота уже подписаны

Логика:
- Лайк = «Интересно, подпишусь»
- Кнопка «Открыть канал» — переход по ссылке
- Отображается в ленте с пометкой «📢 Канал»
- Лимит: 1 канал на пользователя

### 3.5.4 Отображение в ленте
**Приоритет: 🟡 Важно**

- Вкладки в браузинге: `🔍 Люди` | `👥 Группы` | `📢 Каналы`
- Перемешивание в общей ленте (по умолчанию) с пометкой типа
- Групповые и канальные анкеты визуально отличаются (цвет рамки, иконка)
- Фильтр по типу анкеты

### 3.5.5 «Совместимость групп»
**Приоритет: 🟢 Nice to have**

- Бот подсказывает: «Твоя группа про настолки пересекается с группой Маши про D&D — может объединить?»
- Предложение объединить группы с похожими тегами
- Уведомление обоим организаторам

### 3.5.6 Ивенты от групп
**Приоритет: 🟢 Nice to have**

- Группа может создать ивент: «Суббота, 19:00, настолки»
- Бот рассылает анонс всем кто лайкнул эту группу
- RSVP: «Пойду» / «Интересно» / «Не смогу»
- Ивенты отображаются в разделе «Мои группы»

### 3.5.7 «Групповой дейтинг»
**Приоритет: 🟢 Nice to have**

- Два друга создают совместную анкету и ищут пару вместе
- Формат: «Мы — Алексей (INTJ) и Мария (ENFP), ищем ещё пару»
- Отдельный тип анкеты с привязкой двух профилей
- Популярный формат для интровертов

## API Endpoints (текущие + новые)

### Существующие (работают):
- `GET /api/browse` — получить анкеты для просмотра
- `GET /api/profile/me` — мой профиль
- `POST /api/profile/create` — создать анкету
- `POST /api/profile/update` — обновить поле
- `POST /api/profile/publish` — опубликовать
- `POST /api/profile/hide` — скрыть
- `DELETE /api/profile/delete` — удалить
- `POST /api/like` — лайк/дизлайк
- `GET /api/stats` — моя статистика
- `GET /api/who-liked-me` — кто лайкнул (premium)

### Новые (нужно добавить):
